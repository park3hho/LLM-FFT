<|question|>
파이썬에서 리스트와 튜플의 차이는 무엇인가요?
<|answer|>
리스트는 가변(mutable) 자료형으로 값의 수정이 가능하고, 튜플은 불변(immutable) 자료형으로 생성 후 값을 변경할 수 없습니다.

<|question|>
REST API란 무엇인가요?
<|answer|>
REST API는 HTTP 프로토콜을 기반으로 자원을 URI로 표현하고 GET, POST, PUT, DELETE 등의 메서드로 자원을 처리하는 API 설계 방식입니다.

<|question|>
파이썬에서 가상환경을 사용하는 이유는 무엇인가요?
<|answer|>
프로젝트별로 의존성을 분리하여 패키지 충돌을 방지하고, 동일한 개발 환경을 유지하기 위해 가상환경을 사용합니다.

<|question|>
HTTP 상태 코드 404는 무엇을 의미하나요?
<|answer|>
404 상태 코드는 서버에 요청한 리소스가 존재하지 않을 때 반환되는 응답 코드입니다.

<|question|>
클래스와 객체의 차이는 무엇인가요?
<|answer|>
클래스는 객체를 생성하기 위한 설계도이고, 객체는 클래스를 기반으로 생성된 실제 인스턴스입니다.

<|question|>
파이썬에서 예외 처리는 어떻게 하나요?
<|answer|>
try-except 구문을 사용하여 예외가 발생할 수 있는 코드를 감싸고, 예외 발생 시 처리 로직을 정의합니다.

<|question|>
JWT란 무엇인가요?
<|answer|>
JWT는 JSON Web Token의 약자로, 사용자 인증 정보를 JSON 형태로 안전하게 전달하기 위한 토큰 기반 인증 방식입니다.

<|question|>
데이터베이스에서 인덱스의 역할은 무엇인가요?
<|answer|>
인덱스는 데이터 검색 속도를 향상시키기 위한 자료구조로, 조회 성능을 높이는 대신 저장 공간과 쓰기 성능에 비용이 발생합니다.

<|question|>
파이썬에서 딕셔너리를 사용하는 이유는 무엇인가요?
<|answer|>
키-값 구조로 데이터를 빠르게 조회할 수 있어 검색과 매핑 작업에 효율적이기 때문입니다.

<|question|>
OSI 7계층 중 전송 계층의 역할은 무엇인가요?
<|answer|>
전송 계층은 데이터의 신뢰성 있는 전달을 담당하며 TCP와 UDP 프로토콜이 이 계층에 속합니다.

<|question|>
Flask와 Django의 차이점은 무엇인가요?
<|answer|>
Flask는 경량 웹 프레임워크로 유연성이 높고, Django는 다양한 기능이 내장된 풀스택 프레임워크입니다.

<|question|>
파이썬에서 리스트 컴프리헨션이란 무엇인가요?
<|answer|>
리스트 컴프리헨션은 반복문과 조건문을 한 줄로 작성하여 간결하게 리스트를 생성하는 문법입니다.

<|question|>
CORS 에러는 왜 발생하나요?
<|answer|>
브라우저 보안 정책으로 인해 다른 출처(origin)의 리소스 요청이 서버에서 허용되지 않을 때 발생합니다.

<|question|>
CPU와 GPU의 차이점은 무엇인가요?
<|answer|>
CPU는 범용 연산에 최적화되어 있고, GPU는 대규모 병렬 연산에 특화되어 있습니다.

<|question|>
파이썬에서 None은 어떤 의미인가요?
<|answer|>
None은 값이 없음을 나타내는 특수한 객체로, null 개념과 유사하게 사용됩니다.

<|question|>
데이터 직렬화란 무엇인가요?
<|answer|>
데이터 직렬화는 객체를 저장하거나 전송할 수 있는 형태로 변환하는 과정입니다.

<|question|>
GET과 POST 요청의 차이는 무엇인가요?
<|answer|>
GET은 데이터를 조회할 때 사용하고, POST는 데이터를 생성하거나 서버 상태를 변경할 때 사용합니다.

<|question|>
파이썬에서 lambda 함수는 언제 사용하나요?
<|answer|>
간단한 연산을 한 줄로 표현해야 할 때 익명 함수인 lambda를 사용합니다.

<|question|>
캐시(Cache)의 목적은 무엇인가요?
<|answer|>
자주 사용하는 데이터를 임시로 저장하여 응답 속도를 향상시키는 것이 목적입니다.

<|question|>
트랜잭션이란 무엇인가요?
<|answer|>
트랜잭션은 데이터베이스에서 하나의 논리적 작업 단위로, 모두 성공하거나 모두 실패해야 합니다.
